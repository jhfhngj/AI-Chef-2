# AI Chef
Welcome to AI Chef, the cooking simulator for people who want infinite recipes but don’t have infinite money. Forget realistic graphics — this game uses AI to generate whatever monstrosity or masterpiece your ingredients become. With enough VRAM, FLOPS, and questionable judgement, you can create abominations, creations, and rations!

## How do I run it?
To run this, you need a good GPU, a good processor, and lots of RAM. If you have enough to run your model only on GPU, you might get away with having a decent processor. AI Chef uses AI, so that's why.

You also need lots of patience. If you have a decent GPU and a good processor, well, good luck having enough patience to wait that long.

After you finish the painful model downloading, the sluggish weight loading, and the tormenting first prompt, this game will go a bit faster.

The only supported models are AWQ pre-quantized, with the default being a reasonable Qwen 2.5 7B Instruct - AWQ quantized.
We do this so you can stay sane.
